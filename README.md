# 🎮 GameX MERN Platform

A full-stack MERN (MongoDB, Express, React, Node.js) web application featuring a dynamic gaming website, admin dashboard, and backend API for managing content and users.

---

## 🚀 Project Overview

GameX is a complete web platform that includes:

- 🎯 A modern React frontend (user-facing website)
- 🛠️ An admin panel for managing content
- ⚙️ A Node.js + Express backend API
- 🗄️ MongoDB database integration

This project demonstrates a real-world full-stack architecture with separation of frontend, backend, and admin control system.
